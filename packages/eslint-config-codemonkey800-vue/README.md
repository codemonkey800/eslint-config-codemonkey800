# React ESlint Config

This is an ESlint config targeted toward [Vue.js](https://vuejs.org/)
development.

## Setup

To use one of the configs, run:

```sh
$ yarn add -D eslint-config-codemonkey800-vue
```

Then in your `.eslintrc.js`:

```js
module.exports = {
  extends: 'codemonkey800-vue',
};
```
